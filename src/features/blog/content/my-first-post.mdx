## Introduction

Welcome to my first blog post! This post serves as a guide on how to add new content to this portfolio.

## How to Add a Blog Post

To add a new blog post, follow these simple steps:

1.  **Create a new file**: Navigate to the `src/features/blog/content/` directory.
2.  **Naming**: Create a file with the `.mdx` extension (e.g., `my-new-post.mdx`).
3.  **Frontmatter**: At the top of the file, add the metadata between triple dashes `---`.
4.  **Content**: Write your content using standard Markdown syntax. You can also use custom components if they are available in the project.

## Example Frontmatter

```yaml
---
title: Your Post Title
description: A short description of your post.
createdAt: YYYY-MM-DD
updatedAt: YYYY-MM-DD
---
```
